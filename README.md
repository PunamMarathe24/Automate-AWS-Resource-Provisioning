# 🚀 Automate AWS Resource Provisioning using Boto3

## 📌 Project Overview
This project demonstrates how to automate the provisioning of AWS resources using Python and Boto3 instead of manual configuration through the AWS console.

---

## 🎯 Purpose
- Automate infrastructure creation  
- Reduce manual effort and errors  
- Improve deployment speed  
- Enable Infrastructure as Code (IaC) approach  

---

## 🧰 AWS Services Used
- AWS IAM  
- Amazon EC2  
- Amazon S3  
- Boto3 (Python SDK for AWS)  

---

## ⚙️ Features
- 🖥️ Create EC2 instances using script  
- 📦 Create and manage S3 buckets  
- 🔐 Use IAM roles for secure access  
- ⚡ Fully automated infrastructure setup  

---

## 🛠️ Technologies Used
- Python  
- Boto3  
- AWS CLI  

---

## 🚀 How It Works
1. Configure AWS credentials using CLI  
2. Run Python scripts  
3. Scripts interact with AWS using Boto3  
4. Resources are created automatically  

---

## 📌 Conclusion
This project showcases how automation using Boto3 simplifies AWS resource provisioning and enables efficient cloud infrastructure management.
